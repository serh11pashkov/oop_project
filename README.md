# 📦 Система управління складом (Warehouse Management System)

Спрощений навчальний проєкт системи автоматизації складського обліку, розроблений з використанням принципів об'єктно-орієнтованого програмування (ООП). Проєкт містить консольний клієнт та графічний інтерфейс користувача (GUI).

## 🚀 Основний функціонал
- **Каталог товарів (Product)**: реєстрація товарів за категоріями, облік цін та контроль мінімального критичного залишку (дефіциту).
- **База постачальників (Supplier)**: облік контактів та закріплення SKU (товарів) за конкретними постачальниками.
- **Документообіг накладних (Invoice)**: створення прибуткових і видаткових накладних, додавання товарних рядків, проведення документів із автоматичним перерахунком залишків та повний відкат залишків у разі видалення.
- **Збереження даних**: автоматична бінарна серіалізація стану у файл `warehouse-data.bin`.

---

## 🛠 Технологічний стек
- **Мова програмування**: Java 17
- **Графічний інтерфейс**: JavaFX 17 (з темною та світлою CSS-стилізацією)
- **Управління проєктом**: Maven
- **Тестування**: JUnit 5

---

## 💻 Інструкція із запуску

Для збірки та роботи з проєктом потрібен встановлений **JDK 17** та **Maven**.

### 1. Запуск графічного інтерфейсу (JavaFX GUI)
Запуск сучасного GUI з підтримкою світлої теми:
```bash
mvn clean compile javafx:run
```

### 2. Запуск консольного інтерфейсу (CLI)
Для роботи через текстове меню в консолі:
```bash
mvn clean compile exec:java -Dexec.mainClass="edu.university.warehouse.WarehouseApplication"
```

### 3. Запуск модульних тестів (JUnit 5)
Для перевірки працездатності логіки сервісів:
```bash
mvn clean test
```
